# 🎮 Tic Tac Toe React App

A modern and interactive **Tic Tac Toe** game built using **React**, demonstrating state management, React hooks, and clean component structure.  
Players can play X and O, with automatic detection of **winner** or **draw**, and the option to reset the game anytime.

## 🚀 Features

- Interactive **3x3 Tic Tac Toe board**
- Player turn indicator (X / O)
- Winner detection with alerts
- Draw detection with alerts
- Reset game functionality
- Fully responsive design
- Clean and modern UI

---

## 💻 Technologies Used

- **React.js** – Frontend library for building the UI
- **React Hooks** – `useState`, `useEffect` for state and side effects
- **CSS3** – Styling and responsive layout
- **JavaScript (ES6+)** – Logic for game and winner calculation

## ⚙️ React Hooks Used

- **`useState`** – To manage game board state, current player turn
- **`useEffect`** – To handle side effects such as winner or draw detection and alerts
